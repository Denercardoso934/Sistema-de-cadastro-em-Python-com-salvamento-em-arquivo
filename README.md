# Sistema de Cadastro em Python

Sistema simples de cadastro de pessoas desenvolvido em Python, com salvamento de dados em arquivo.

## 📋 Funcionalidades

- Cadastrar pessoas com nome e idade
- Listar todos os cadastrados
- Dados salvos em arquivo (persistência)

## 🚀 Como usar

1. Clone o repositório
2. Execute o arquivo `cadastro.py`
3. Escolha uma opção do menu

## 🛠️ Tecnologias

- Python 3.12
